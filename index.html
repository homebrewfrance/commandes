Recherche d'applications

	   
    
	   
  • .udb [nom de l'app] : recherche une application dans la base de données Universal-DB (Universal-Updater).
  • 
	   
  • .gamebrew [nom de l'app] : recherche une application dans la base de données Gamebrew.
    • 
		

	   
Nintendo Switch

	   
    
		   
  • /imsp |NUMERO DE SERIE SWITCH| : vérifie si votre Switch est compatible avec la faille RCM.
    • 
		   
  • -nxcheck : affiche où se situe la numéro de série de votre Nintendo Switch.
    • 
		   
  • -ftpdswitchandroid : affiche comment configurer FTPD sur votre Nintendo Switch avec votre appareil Android.
    • 
		   
  • -ftpdswitchwindows : affiche comment configurer FTPD sur votre Nintendo Switch avec votre ordinateur sous Windows.
